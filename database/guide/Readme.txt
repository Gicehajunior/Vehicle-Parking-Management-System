How to run the Vehicle Parking Management System Project

>Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)
>Open PHPMyAdmin (http://localhost/phpmyadmin)
>Create a database with name vpmsdb
>Import vpmsdb.sql file(given inside the the database folder in  sql file)
>Run the script http://localhost/vehicle_parking (frontend)
